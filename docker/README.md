# Docker

A practical guide to learning and implementing **Docker** using hands-on labs.

This documentation is based on a real lab environment created using **Docker Desktop** and **Docker Engine** on Linux.

---

## Lab Environment

| Component | Details |
|----------|---------|
| Platform | Docker Engine |
| Operating System | Ubuntu 24.04 / Rocky Linux 9 |
| Docker | Latest Stable Version |
| Registry | Docker Hub |
| Editor | Visual Studio Code |
| Terminal | Bash |

## Learning Outcomes

After completing this documentation, you will be able to:

- Understand Docker architecture
- Build and run Docker containers
- Create custom Docker images
- Manage persistent storage using volumes
- Configure Docker networking
- Write Dockerfiles
- Deploy applications using Docker Compose
- Monitor and troubleshoot containers
- Apply Docker best practices

---

## Prerequisites

- Basic Linux commands
- Basic terminal knowledge
- A system with Docker installed
