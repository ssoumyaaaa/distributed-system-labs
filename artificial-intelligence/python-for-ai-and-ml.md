# Python for Artificial Intelligence and Machine Learning

## Overview

Python is the most popular programming language for Artificial Intelligence (AI) and Machine Learning (ML). It is simple to learn, easy to read, and provides powerful libraries that help developers build intelligent applications with less code.

Today, Python is widely used by data scientists, AI engineers, researchers, and software developers around the world.

## Installing Python

Download Python from the official website:

**https://www.python.org/downloads/**

Verify the installation:

```bash
python --version
```

Expected output:

```text
Python 3.x.x
```

---

## Your First Python Program

Create a file named **hello.py**.

```python
print("Hello, Artificial Intelligence!")
```

Run the program:

```bash
python hello.py
```

Output:

```text
Hello, Artificial Intelligence!
```
