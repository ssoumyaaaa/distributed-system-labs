# Unsupervised Learning

## Overview

Unsupervised Learning is a type of Machine Learning where the model learns from **unlabeled data**. Unlike Supervised Learning, there are no correct answers or labels provided. The model analyzes the data to discover hidden patterns, similarities, or groups on its own.

For example, an online shopping website can group customers with similar buying habits without knowing anything about them beforehand.

---

## Why Learn Unsupervised Learning?

- Discovers hidden patterns in data
- Groups similar data automatically
- Helps understand large datasets
- Identifies unusual or abnormal data
- Useful when labeled data is unavailable

---

## Types of Unsupervised Learning

### Clustering

Clustering groups similar data points together.

Examples:

- Customer Segmentation
- Product Categorization
- News Article Grouping
- Social Network Analysis

---

### Association

Association finds relationships between different items.

Examples:

- Market Basket Analysis
- Product Recommendations
- Frequently Bought Together
- Shopping Pattern Analysis
