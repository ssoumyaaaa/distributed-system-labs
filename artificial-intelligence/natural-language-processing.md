# Natural Language Processing (NLP)

## Overview

Natural Language Processing (NLP) is a branch of Artificial Intelligence that enables computers to understand, interpret, and generate human language. It combines AI, Machine Learning, and linguistics to help computers communicate with people using text or speech.

NLP powers many everyday applications such as chatbots, virtual assistants, language translation, and email filtering.

---

## Why Learn NLP?

- Enables computers to understand human language
- Automates text and speech processing
- Improves customer support with chatbots
- Supports language translation
- Powers many modern AI applications

---

## How NLP Works

```text
Input Text or Speech
         │
         ▼
Text Processing
         │
         ▼
Language Understanding
         │
         ▼
Generate Response
         │
         ▼
Output
```

## Common NLP Tasks

### Text Classification

Classifies text into predefined categories.

Examples:

- Spam Detection
- Sentiment Analysis
- News Classification

---

### Language Translation

Converts text from one language to another.

Examples:

- English → French
- English → Hindi

---

### Question Answering

Answers questions based on the provided information.

Examples:

- ChatGPT
- Customer Support Bots

---

### Text Summarization

Creates a shorter version of a long document while preserving the main points.

Examples:

- News Summaries
- Document Summaries
