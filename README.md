# distributed-system-labs
Learning repository for distributed systems, including GemFire, Kubernetes, RabbitMQ, VMware, Docker, and related technologies.


## 📖 About

This repository contains practical learning materials, setup guides, architecture diagrams, and troubleshooting notes for technologies commonly used in distributed systems.

The goal of this repository is to document real-world hands-on experience and serve as a reference for learning and experimentation.

## 🛠️ Technologies

- Apache Geode (GemFire)
- Kubernetes
- Docker
- Podman
- RabbitMQ
- VMware Workstation
- Linux (Rocky Linux & Ubuntu)
- Networking
- Git & GitHub
