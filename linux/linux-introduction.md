# Linux Introduction

Linux is a free and open-source operating system widely used for servers, cloud platforms, containers, networking, and DevOps. Most modern technologies such as Docker, Kubernetes, RabbitMQ, Kafka, and Redis run on Linux.

---

## Why Learn Linux?

- Open source
- Secure and stable
- High performance
- Widely used in cloud environments
- Preferred operating system for DevOps
- Supports automation and scripting

## Linux Architecture

```text
+----------------------+
|      Applications    |
+----------------------+
|         Shell        |
+----------------------+
|        Kernel        |
+----------------------+
|      Hardware        |
+----------------------+
```

### Hardware
Physical components like CPU, RAM, disk, and network devices.

### Kernel
The core of Linux that manages hardware, memory, processes, and devices.

### Shell
The command-line interface that allows users to interact with the operating system.

### Applications
Programs such as Docker, Git, Nginx, Java, and Python.

## Linux vs Windows

| Linux | Windows |
|--------|----------|
| Open source | Proprietary |
| CLI focused | GUI focused |
| Highly customizable | Limited customization |
| Preferred for servers | Common for desktops |
| Strong scripting support | PowerShell and CMD |
